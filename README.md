# ExBitflyer

Bitflyer API Client for Elixir

## Installation

Add the `ex_bitflyer` package to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [{:ex_bitflyer, "~> 0.0.1"}]
end
```
